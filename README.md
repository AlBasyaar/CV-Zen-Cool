## ✨ Features

### 🤖 AI-Powered Content Assistance

Generate professional resume content, improve writing quality, and receive intelligent suggestions powered by advanced AI models.

### 🎨 Modern & Responsive UI

Built with Next.js and Tailwind CSS, providing a clean and responsive experience across desktop and mobile devices.

### ✏️ Intuitive CV Editor

Easily create, edit, and organize resume sections with a distraction-free interface.

### 👁️ Real-Time Preview

Instantly preview changes while editing your CV.

### 📄 Professional Templates

Choose from clean and modern templates optimized for recruiters and ATS systems.

### 📥 Export to PDF

Download your completed CV in a professional PDF format.

## 📖 Overview

CV-Zen-Cool is a modern resume builder that helps users create professional CVs quickly and efficiently.

Built using Next.js, TypeScript, and Tailwind CSS, the platform focuses on delivering a smooth user experience while integrating AI-powered assistance for content generation, resume optimization, and personalized recommendations.

The goal of CV-Zen-Cool is simple:

> Create better resumes with less effort.

## 🚀 Quick Start

### Prerequisites

* Node.js 20+
* npm

### Installation

```bash
git clone https://github.com/AlBasyaar/CV-Zen-Cool.git

cd CV-Zen-Cool

npm install
```

### Environment Variables

Create a `.env` file:

```env
CLAUDE_API_KEY=your_api_key
NEXT_PUBLIC_APP_ENV=development
```

### Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
